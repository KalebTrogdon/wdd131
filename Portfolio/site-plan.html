Kaleb's Developer Portfolio - Site Plan

Home Projects

Purpose

This site showcases my personal coding projects, links to my social and professional profiles, and gives visitors insight into my work and skills as a developer.

Audience

This site is intended for hiring managers, potential clients, collaborators, and fellow developers who are interested in my work or want to reach out to me professionally.

Dynamic Elements

JavaScript will be used to load projects from a JSON file, apply category filtering, and display a dynamic greeting message based on the current time. A contact form will include real-time validation.

Logo

Colors and Fonts

  • Primary Color: #1f2937 (Dark Gray)
  • Accent Color: #3b82f6 (Blue)
  • Background Color: #f9fafb (Off-White)
  • Font: 'Inter', sans-serif

Content

The home page will feature a welcome message, intro about me, links to socials, and preview of featured projects. The projects page will list all projects with descriptions, tags, and links to GitHub/live demos.
